Terminologies

Tag/Elements : String enclosed in < and> is a tag/element in html. Html file has opening tag and closing tag. An opening tag is enclosed in <> and a closing tag is enclosed in </ and > . For example : <html> </html>

Attributes define additional characteristics or properties of the element such as width and height of an image. Attributes are always specified in the start tag (or opening tag) and usually consists of name/value pairs like name="value". Attribute values should always be enclosed in quotation marks.

What is html, Why html is needed ?

  1. HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.
  2. "Hypertext" refers to links that connect web pages to one another, either within a single website or between websites.
  3. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.
Lets understand it like this, for anything on internet you need some type of method where you can write you content. Also to view that content we need a tool where we can open that content.
Html is just like any other file on the system. It have the extension of html, htm for example index.html or index.htm index.html (perviously html was not supported by DOS operating systems thus htm was introduce, but now all OS support html).

Lat's start understanding the code

  1. DOCTYPE : All HTML documents must start with a document type declaration: .

    The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly.

    It must only appear once, at the top of the page (before any HTML tags).

    The <!DOCTYPE> declaration is not case sensitive.

  2. HTML: The HTML document itself begins with <html> and ends with </html>.
  3. lang="en":The HTML lang attribute is used to identify the language of text content on the web .
    <html lang="en">.
    ....
    </html>.
  4. head: The tag in HTML is used to define the head portion of the document which contains information related to the document.

    The tag contains other head elements such as <meta>, <title>, <link>, <style> etc.

    • Meta: The <meta> tag is used to provide such additional information.

      This tag is an empty element and so does not have a closing tag but it carries information within its attributes.

    • Title: The HTML <title>tag is used for indicating the title of the HTML document.

      The body title is placed between the <head> and the <head> tags.

  5. body: The contents of the <body> element are what actually get displayed to the user visiting your web page or viewing your document.

    And yet, it is very simple.

    The <body> must be a direct child of <html>, it must come after <head> (if <head> is present, which it doesn’t have to be), there can be only one of it, and … that’s it.

    <body>
    .....
    </body>

    HTML Elements

    The basic elements of an HTML page are:
  1. The <header> element represents a container for introductory content or a set of navigational links.
      2. A text header, denoted using the <h1>, <h2>, <h3>, <h4>, <h5>, <h6> tags.
  2. A paragraph, denoted using the <p> tag.
  3. A horizontal ruler, denoted using the <hr> tag.
  4. A link, denoted using the <a> (anchor) tag.
  5. A list, denoted using the <ul> (unordered list), <ol> (ordered list) and <li> (list element) tags.
  6. An image, denoted using the <img> tag
  7. A divider, denoted using the <div>tag
  8. A text span, denoted using the <span> tag
  9. The <iframe> tag specifies an inline frame.
  10. The <section> tag defines a section in a document.
  11. lebal
  12. The <b> tag specifies bold text without any extra importance.
  13. The <em> element is used to indicate text that should receive greater emphasis than the surrounding text.
  14. The <u> element was originally used to identify text that should be underlined.
  15. In HTML, the <br> tag is used for line break. It is an empty tag i.e. no need to add an end tag. Writing
    tag is perfectly fine.
  16. The HTML <table> tag is used for defining a table. The table tag contains other tags that define the structure of the table.
      17. The <tr> tag defines a row in an HTML table.
      The <th> tag defines a header cell in an HTML table.
      The <td> tag defines a standard data cell in an HTML table.
  17. The <blockquote> tag specifies a section that is quoted from another source.
  18. The <dl> tag defines a description list.
      19. The <dt> tag defines a term/name in a description list.
      The <dd> tag is used to describe a term/name in a description list.
  19. The <strong> tag is used to define text with strong importance.
  20. The <del> tag defines text that has been deleted from a document.
  21. The <ins> tag defines a text that has been inserted into a document.
  22. The <q> tag defines a short quotation.
  23. The <datalist> tag specifies a list of pre-defined options for an <input> element.
      The <summary> tag defines a visible heading for the <details> element. The heading can be clicked to view/hide the details.
  24. The <dialog> tag defines a dialog box or subwindow.
  25. The <form> tag is used to create an HTML form for user input.
      The <fieldset> tag is used to group related elements in a form.
      The <legend> tag defines a caption for the <fieldset> element.
      The <input> tag specifies an input field where the user can enter data.
      The <output> tag is used to represent the result of a calculation (like one performed by a script).
  26. The <small> tag defines smaller text (like copyright and other side-comments).
  27. The <sub> tag defines subscript text.
  28. The <sup> tag defines superscript text.
  29. The <textarea> tag defines a multi-line text input control.
  30. The <template> tag is used as a container to hold some HTML content hidden from the user when the page loads.
  31. The <progress> tag represents the completion progress of a task.
  32. The <pre> tag defines preformatted text.
  33. The <samp> tag is used to define sample output from a computer program.
  34. The <base> tag specifies the base URL and/or target for all relative URLs in a document.
  35. The <address> tag defines the contact information for the author/owner of a document or an article.
  36. The <link> tag defines the relationship between the current document and an external resource.
  37. The <button> tag defines a clickable button.
  38. The <select> element is used to create a drop-down list.
      The <option> tag defines an option in a select list.
  39. The <map> tag is used to define an image-map. An image-map is an image with clickable areas.
      The <area> tag defines an area inside an image-map (an image-map is an image with clickable areas).
  40. The <footer> tag defines a footer for a document or section.
  41. The <object> tag defines a container for an external resource.
  42. The <nav> tag defines a set of navigation links.
  43. The <mark> tag defines text that should be marked or highlighted.
  44. The <meter> tag defines a scalar measurement within a known range, or a fractional value.
  45. The <kbd> tag is used to define keyboard input.
  46. The <source> tag is used to specify multiple media resources for media elements, such as <video>, <audio>, and <picture>.
      The <audio> tag is used to embed sound content in a document, such as music or other audio streams.
      The <video> tag is used to embed video content in a document, such as a movie clip or other video streams.
      The <picture> tag gives web developers more flexibility in specifying image resources.
